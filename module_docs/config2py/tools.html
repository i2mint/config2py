config2py.tools¶

Various tools

config2py.tools.extract_exports(exports: str) → dict[source]¶

Get a dict of {name: value} pairs from the name="value" pairs of unix export lines (that is, lines of the ``export NAME="VALUE" format

Parameters

exports – Filepath or string contents thereof

Returns

A dict of extracted {name: value} pairs

>>> extract_exports('export KEY="secret"\nexport TOKEN="arbitrary"')
{'KEY': 'secret', 'TOKEN': 'arbitrary'}

You have access to environment variables through os.environ, but if you want to extract exports from only a specific file (env vars are often placed in different linked files), or the exports are defined in a string you hold, then this simple parser can be useful.
